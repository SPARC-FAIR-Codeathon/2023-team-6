script.py:
```python
import pandas as pd
from sklearn.linear_model import LinearRegression

# Load the data
data = pd.read_csv('data/datafile.csv')

# Split the data into features and target
X = data.drop('y', axis=1)
y = data['y']

# Train a linear regression model
model = LinearRegression()
model.fit(X, y)

# Make predictions
predictions = model.predict(X)

# Save the results
results = pd.DataFrame({'Predicted': predictions})
results.to_csv('results/results.txt', index=False)

# Record package versions
import subprocess
with open("requirements.txt", "w") as f:
    subprocess.run(["pip", "freeze"], stdout=f)
```

Dockerfile:
```
FROM python:3.9

WORKDIR /app

COPY script.py /app/script.py

RUN pip install scikit-learn pandas

CMD [ "python", "script.py" ]
```

To run the Docker container, use the following command:
```
docker run -v /path/to/data:/app/data -v /path/to/results:/app/results image_name
```
Replace `/path/to/data` with the actual path to the "data" folder on your host machine, and `/path/to/results` with the actual path to the "results" folder on your host machine. Replace `image_name` with the desired name for the Docker image.
